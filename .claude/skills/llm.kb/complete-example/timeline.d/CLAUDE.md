# Timeline Directory - Agent Guide

## What Belongs Here

Schedule and logistics for party day:
- Setup tasks and timing
- Event phases
- Cleanup plan

## What Does NOT Belong

- Activity rules (those go in `games.d/`)
- Food details (those go in `food.d/`)

## Time Blocks

Party runs 2:00 PM - 5:00 PM.
Setup starts at 12:00 PM.
