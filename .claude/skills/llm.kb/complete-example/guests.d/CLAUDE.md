# Guests Directory - Agent Guide

## What Belongs Here

One file per invited child containing:
- RSVP status
- Dietary restrictions/allergies
- Parent contact info
- Any special considerations

## What Does NOT Belong

- Adult guests (this is kids only)
- Gift tracking (separate concern)
- Seating arrangements (that's logistics)

## When to Update

- New invitation sent → create file with `rsvp: pending`
- RSVP received → update status
- New allergy info → add to file, flag in root CLAUDE.md
