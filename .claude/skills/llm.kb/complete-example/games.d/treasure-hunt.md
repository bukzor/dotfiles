# Treasure Hunt

Kids follow clues hidden around the yard to find the prize box.

## Setup Required

1. Write 8 clue cards
2. Hide clues in sequence
3. Place prize box at final location
4. Brief parents on boundaries

## Clue Locations (Outdoor)

1. Mailbox → points to big tree
2. Big tree → points to garden hose
3. ...

## Rainy Day Version

Move indoors with adapted locations:
1. Coat closet → bookshelf
2. ...

## Prize Box Contents

- Small toys for everyone (not just winner)
- Stickers
- Candy (peanut-free verified)
