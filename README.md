Buck's Dotfiles
===============

These are my dotfiles. There are many like them, but these are mine.

Quick setup
-----------

Copy-paste to a terminal:

    git clone --no-checkout git@github.com:bukzor/dotfiles.git  tmp-dotfiles
    mv tmp-dotfiles/.git .
    rmdir tmp-dotfiles
    git checkout  -f
    git submodule update --init
